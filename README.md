# Wikimedia Article Viewer

This project streamlines access to articles in your selected language by seamlessly integrating with the Wikimedia API. Explore tailored content that caters to your linguistic preferences, enhancing your browsing experience.

## Technologies used
- Vue.js
- @wikimedia/codex (for components)
- Wikimedia API

## Installation

Run the development server

```sh
npm run dev
```
